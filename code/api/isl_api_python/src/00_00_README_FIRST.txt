Before starting to work on the development of the ISL Python API,
1. ISL shall be built using CMake, and installed in the <ISL repository>/install folder
2. Python 3.8.10 or higher version shall be installed
3. A virtual environment of Python 3.8.10 (or higher version) shall be created using the 00_create_venv script (edit the script to correctly set the Python installation path to use)
4. Update the virtual environnement using the script 01_update_venv
