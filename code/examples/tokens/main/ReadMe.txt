To execute the co-simulation, double-click on the run.bat file.

If the execution is successfull, you should see the 4 lines below in the run.bat command window:
---------- PROCESS1\PROCESS1.LOG: 1
Info: all processes/token1 has been successfully executed

---------- PROCESS1\PROCESS1.LOG: 1
Info: all processes/token2 has been successfully executed

Note: If you move the tokens example folder to another location, you must update the OPENISL_PATH environment variable from the run.bat file
