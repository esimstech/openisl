Usage instructions:
------------------

1. Copy the files "modelDescription.xml", "buildislfmu.bat" and "islfmuenv.bat" from <OpenISL path>\connect\fmi to the folder of your choice.
2. Edit the file "islfmuenv.bat" and update the variables
3. Edit the modelDescription.xml
	3.1. Update model name and identifiers
	3.2. Update the times: start, end and step size
	3.3. Add your inputs and outputs below the ISL parameters
4. Build a new XML file based on the modelDescription.xml
	4.1. Copy the inputs and outputs variables from the modelDescription.xml
	4.2. Do not copy ISL parameters (with reference numbers 90000 and more)
	4.3. Inputs of the FMU shall be outputs in the XML file. If the modelDescription has an input with a reference number "10", the XML shall have an output with the attribute "name" with the value "10".
	4.4. Outputs of the FMU shall be inputs in the XML file. If the modelDescription has an output with a reference number "21", the FMX shall have an input with the attribute "name" with the value "21".
5. Build the FMU using the "buildislfmu.bat" script
6. Use the ISL FMU with any software used in FMI 2.0 master mode (co-simulation). 
	6.1. Update the ISL parameters
		6.1.1. 90000: Working directory
		6.1.2. 90001: XML file name and path (relative to the working directory). Absolute file names are not allowed
		6.1.3. 90002: ISL session identifier
